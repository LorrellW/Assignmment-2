    C++/WinRT Assignment 2 Linked List
========================================================================

The LinkedList class contains several methods for manipulating the linked list:

push_front(int data): Adds a new node with the specified data value to the front of the list.
push_back(int data): Adds a new node with the specified data value to the back of the list.
pop_front(): Removes the node at the front of the list.
pop_back(): Removes the node at the back of the list.
front_back(): Prints the values at the front and back of the list.
empty(): Returns true if the list is empty, false otherwise.
remove(size_t index): Removes the node at the specified index in the list.
insert(size_t index, const int& item): Inserts a new node with the specified data value at the specified index in the list.
find(const int& item): Finds the index of the first node in the list with the specified data value.
printlist(): Prints the data values of all nodes in the list.


To use the linked list, create an instance of the LinkedList class and call 
the desired methods on that instance.
